 ## API:
 
 ```javascript
 zalgoize.chars = {
     up: [...],
     down: [...],
     middle: [...]
 };
 ```
 As a convenience, the chars used for the top, bottom and middle are available here.
 
 ```javascript
 zalgoize.encode(string, options);
 ```
